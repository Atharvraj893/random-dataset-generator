# random-dataset-generator
The Synthetic Dataset Generator is a Python-based tool designed for students and beginners to overcome the challenge of finding clean, structured data for machine learning. By utilizing libraries like pandas, numpy, and random, users can generate meaningful, custom datasets that simulate real-world scenarios.

The project supports multiple dataset types such as employee, student, and sales datasets. Each dataset is generated using logical relationships between variables to simulate real-world scenarios. For example, salary depends on experience, marks depend on study hours, and revenue depends on advertising spend, making the data more meaningful.

Users can control the number of rows and dataset type, making the tool flexible for different use cases.For now The generator give you a clean data or else you can clean data.

The project allows exporting datasets in CSV and Excel formats, making it easy to use the data in tools like Python, Power BI, or Excel. It is built using simple libraries like pandas, numpy, and random, keeping it beginner-friendly.
